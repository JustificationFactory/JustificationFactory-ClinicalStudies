GUI for clinical studies analysis using Justification Factory

## Dependencies needed

You need this set of tools :
* Java 8 (Oracle version)
* Maven

## To run the platform

* Compile and run tests ``` mvn clean install```
* Run GUI ```mvn exec:java```
